# Dental Clinic Appointment System - Spring Boot Backend Modulleri

- Java 21
- Spring Boot 4.0.0


## 1. Authentication & Authorization Module

-   Spring Security + JWT
-   Role-based access
-   Token management

## 2. User Management Module

-   Patient, Dentist, Staff yonetimi
-   Profil bilgileri
-   Klinik register -login islemleri klinik staff ,admin,doktor birden fazla kullanici aylik hesaba etki
-   Admin yonetimi sisteme klinik ekleme, silme(crud islemleri)

## 3. Appointment Scheduling Module

-   Randevu olusturma, guncelleme, iptal
-   Cakisma kontrolu
-   Randevu durumlari

## 4. Dentist Availability & Calendar Module

-   Calisma saatleri
-   Tatil/izin gunleri
-   Takvim hesaplamalari

## 5. Treatment / Services Module

-   Tedavi listesi
-   Sure ve fiyat bilgileri

## 6. Patient Medical Records Module

-   Hastanin gecmis tedavileri
-   Tibbi notlar ve kayitlar

## 7. Billing & Payments Module (Opsiyonel)

-   Fatura, odeme durumu

## 8. Notifications Module

-   Email / SMS bildirimleri

## 9. Inventory & Supplies Module (Opsiyonel)

-   Stok takibi
-   Minimum stok uyarilari

## 10. Reports & Dashboard Module

-   Randevu ve gelir raporlari
-   Yonetimsel istatistikler

## Teknik Mimari:

-   Controller, Service, Repository katmanlari
-   Spring Data JPA
-   Swagger API Docs
-   Flyway
-   Metric-grafana-prometheus
-   Docker compose
-   Redis
